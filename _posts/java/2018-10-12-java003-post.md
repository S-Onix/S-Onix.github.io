## 자료형
- 변수 또는 상수가 차지하는 메모리의 양(byte)와 종류를 의미함
- 자바의 자료형은 기본형과 참조형으로 나뉜다.

### 기본형
- 실제 값(data)을 저장
  - boolean(true, false) : 8bit(1Byte)
  - byte : 8bit
  - short : 16bit
  - char : 16bit
  - int : 32bit
  - long : 64bit
  - float : 32bit
  - double : 64bit

  |종류/크기|1 byte|2 byte|4 byte|8 byte|
  |---|---|---|---|---|
  |논리형|booelan||||
  |문자형||char|||
  |정수형|byte|short|int|long|
  |실수형|||float|double|


### 참조형
- 값이 저장되어 있는 주소의 값을 저장
  - 클래스(class)
  - 배열(array)
  - 인터페이스(interface)
  - 열거형(enum)
